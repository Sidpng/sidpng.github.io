# sidpng.github.io

Personal portfolio site for **Siddhartha Upadhyay** — Senior QA Engineer / SDET.

A single-page, dependency-free static site (HTML + CSS + vanilla JS) with scroll-reveal animations,
animated counters, skill meters, an experience timeline, a particle background, and a 3D-tilt project
grid. Deployed via GitHub Pages → **https://sidpng.github.io**.

## Edit
- Content: `index.html`
- Styling/animations: `styles.css`
- Interactions: `script.js`

## Personal info placeholders
Search `index.html` for `data-placeholder="..."` — email, phone, linkedin, location, resume are left
blank until filled in.
